## 📚 Features

- 🔊 Convert text to spoken audio using browser's SpeechSynthesis API
- 🎨 Clean and responsive UI with Tailwind CSS
- ⚡ Fast development experience using Vite
- 🌙 Light/dark mode ready (optional extension)
